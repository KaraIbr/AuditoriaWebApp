# 01. Introducción

## Propósito

WebAudit es un framework estandarizado para auditar aplicaciones web, e-commerce, APIs y software interno. Proporciona una metodología repetible y documentada basada en estándares OWASP.

## ¿Para quién?

- **Auditores de seguridad** — Framework completo con checklists y herramientas
- **Desarrolladores** — Guía para revisar su propio código
- **Equipos de seguridad** — Proceso estandarizado para auditorías internas
- **Consultores** — Base metodológica para entregar a clientes

## ¿Por qué WebAudit?

- **Basado en estándares** — OWASP Top 10, ASVS, WSTG, NIST
- **Estructurado** — Misma metodología para todas las auditorías
- **Documentado** — Cada hallazgo tiene evidencia y trazabilidad
- **Open source** — Sin vendor lock-in, libre de usar y modificar
- **Branch workflow** — Cada auditoría en su branch, historial completo

## Cómo usar este framework

1. Clonar el repositorio
2. Crear una branch para la auditoría
3. Seguir las fases del pipeline
4. Completar checklists según el tipo de app
5. Documentar hallazgos
6. Generar reporte final
