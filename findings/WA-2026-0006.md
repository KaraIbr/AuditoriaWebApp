# Default JWT secret key inseguro

**ID:** WA-2026-0006
**Severidad:** Low
**OWASP Top 10:** A02 — Cryptographic Failures
**OWASP WSTG:** WSTG-CRYP-04
**ASVS:** 2.1.2
**Estado:** Open
**Fecha:** 2026-07-07
**Auditor:** Security Team

---

## Descripción

El valor por defecto de `IAM_JWT_SECRET_KEY` en desarrollo es `"change-me-in-local-development"`. Si este valor no se cambia en producción, cualquier persona que conozca el código fuente puede firmar JWTs válidos.

## Impacto

Un atacante puede forjar JWTs arbitrarios y autenticarse como cualquier usuario, incluyendo administradores.

## Probabilidad

Baja — Requiere que el secret no se haya cambiado en producción.

## Condición

- `IAM_JWT_SECRET_KEY` no configurado en producción
- El atacante conoce el código fuente (repositorio público o filtrado)

## Evidencia

`IAM/src/iam/core/config.py:32`:
```python
jwt_secret_key: SecretStr = Field(
    default=SecretStr("change-me-in-local-development"),
    min_length=16,
)
```

## Recomendación

- Cambiar el secret JWT antes de cualquier deploy a producción
- Usar un secret manager o variable de entorno segura
- Rotar el secret periódicamente
